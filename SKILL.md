---
技能名称: 珠宝电商全案生成技能
技能描述: 珠宝电商图片、文案与短视频一站式生成助手。用户只需输入珠宝产品和简单描述，即可按预设场景自动产出商品图、卖点文案、详情页文案、短视频脚本与成片素材。支持亚马逊 9 图 + 10 秒视频、Shopify PDP、天猫/淘宝详情页、小红书/抖音种草短片等电商物料。当用户提到“珠宝电商图”“亚马逊九图”“商品图文案视频一起出”“戒指上架素材”“项链 10 秒视频”“珠宝全套电商物料”等需求时使用。
---

技能内容：

本技能是一个**复合型 skill**，目标是把“珠宝产品 + 简单描述”转化为可用于电商上架、广告投放和社媒种草的完整物料包。

默认不要求用户填写复杂表单。只要用户给出珠宝产品和一句简单描述，就应自动补齐合理默认值并进入执行；只有缺少“产品是什么”这类无法推断的关键字段时才打断流程。

## 一、能力边界

### 1.1 本技能能做什么

- 生成珠宝商品主图、角度图、细节图、佩戴图、场景图、包装图和卖点图。
- 生成适配电商平台的标题、五点卖点、短描述、详情页模块文案、视频旁白/字幕文案。
- 生成短视频脚本、关键帧、镜头运动提示词，并调用即梦视频工具生成 4-15 秒产品视频。
- 对已有珠宝产品图进行统一风格改造、去背景、白底化、高清放大、参考图驱动视频。
- 按预设场景一次性交付完整物料，例如“亚马逊 9 图 + 10 秒视频”。

### 1.2 本技能不应做什么

- 不编造宝石证书、克拉数、产地、净度、价格、库存、品牌授权或医疗功效。
- 不把风格参考当成可复制的品牌原款；除非用户上传的是自有产品图，否则只做“风格启发”。
- 不在缺少产品图片时承诺与真实 SKU 完全一致；应标注为“设计渲染版本”。
- 不让用户理解工具名或提供 JSON；工具调用是技能内部流程。
- 不把九宫格理解成九个随机设计。亚马逊 9 图是同一 SKU 的九个电商表达槽位。

### 1.3 即梦工具清单映射

本技能必须按即梦在线版文档中的工具能力组织流程。不同工具在珠宝电商全案中的职责如下：

| 工具名 | 本技能中的用途 | 默认策略 |
|---|---|---|
| `text2image` | 无产品图时生成产品母图、概念首图、详情页首屏视觉 | 可用，但生成后必须建立产品身份锁 |
| `image2image` | 基于产品图或母图生成主图、角度图、细节图、佩戴图、场景图、包装图 | 核心图片工具，优先使用 |
| `foreground_segmentation` | 白底图、透明底素材、详情页产品抠图 | 用于标准化和组合，不用于修复结构错误 |
| `image_super_resolution` | 主图、细节图、视频关键帧高清放大 | 只对结构正确的图使用 |
| `text2video` | 无任何参考图且用户只要快速概念视频时使用 | 非默认；珠宝电商成片优先参考图驱动 |
| `image2video` | 用 1 张主图生成简单动态视频 | 适合快速 4-10 秒产品动效 |
| `start_end2video` | 用首尾帧控制开盒、远近推进、前后对比 | 适合明确转场 |
| `multi_frame2video` | 用多张商品图/佩戴图串成短片 | Amazon 9 图视频可用 |
| `multi_modal2video` | 用图片、视频、音频等多模态素材生成高一致性广告片 | 默认优先的视频工具 |
| `video_editor` | 拼接片段、配乐、整理最终成片 | 最终合成工具 |
| `generate_form_for_info_collection` | 缺少必填信息时收集最小字段 | 只在必填缺失时调用 |
| `creation_agent_search` | 检索平台规范、趋势参考、非侵权风格参考 | 可选；不得用搜索结果伪造品牌授权 |

### 1.4 即梦在线版适配规则

即梦在线版与本地 Dreamina CLI 不是同一套调用语境。本技能按在线 skill 说明编写，执行时遵守以下规则：

- 工具名称以在线版实际暴露能力为准。文档速查表使用 `text2image`、`image2image` 等通用名；完整示例中出现 `text2image_v3`、`image2image_v3`。如果在线环境提供 v3 工具，实际调用优先使用 v3；如果只提供通用名，则使用通用名。
- 不在 skill 中硬编码本地 CLI 模型名、账号命令、提交 ID 查询命令或下载参数。在线版模型/预设由即梦运行环境选择或在工具面板中配置。
- 预设能力优先来自即梦文档的典型工具链路：图片驱动成片、直出视频、已有素材二次创作、多素材视频、素材净化高清交付、先搜索再生成。
- 珠宝电商成片默认走“图片驱动成片”或“多素材视频”，因为产品身份比纯氛围更重要。
- `text2video` 只作为快速概念视频分支，不作为有产品图、有母图或已生成商品图时的默认成片方式。
- 当在线工具提供内置风格、比例、清晰度、视频时长等预设时，优先选择最贴近当前场景的商业摄影、电商主图、产品广告、竖屏短视频、高清输出类预设；不要为了追求风格牺牲产品一致性。

### 1.5 典型工具链路迁移

以下链路直接对应即梦在线版文档中的“典型工具链路”，并转译为珠宝电商场景：

| 即梦典型链路 | 珠宝电商用法 | 本技能预设 |
|---|---|---|
| `generate_form_for_info_collection` → `text2image` → `image2video` → `video_editor` | 用户没有产品图时，先补齐最小 brief，生成产品母图，再做简单短视频并合成 | 快速概念上架包 |
| `generate_form_for_info_collection` → `text2video` → `video_editor` | 仅做气氛概念片或早期方向演示，不要求 SKU 一致 | 非默认概念视频 |
| `image2image` → `image2video` → `video_editor` | 用户已有产品图时，先生成商品图/佩戴图，再把主图做动态展示 | `clean_product_pack`、轻量视频 |
| `text2image` / 已有素材 → `multi_modal2video` → `video_editor` | 用主图、佩戴图、礼盒图、音频等做高一致性 10 秒广告 | `amazon_9_images_10s_video` 默认视频 |
| `image2image` → `foreground_segmentation` → `image_super_resolution` | 产品图净化、白底主图、透明底素材、详情页高清图 | `clean_product_pack` |
| `creation_agent_search` → `text2image` | 需要先查平台规范、趋势参考或非侵权风格参考，再生成概念图 | 风格/平台不明确时 |

## 二、交互阶段

交互阶段只负责提取、补齐、确认最小信息。默认走“一句话启动”，不做冗长问卷。

### 步骤 1：信息提取

从用户输入中提取以下字段：

| 字段 | 必填 | 缺失时处理 |
|---|---|---|
| 珠宝产品 | 是 | 如果无法判断是戒指、项链、耳环、手链、吊坠、胸针等，提示用户补充，停止流程 |
| 简单描述 | 是 | 如果只有“做一套图”但没有风格、材质、用途或外观线索，提示用户补充一句描述，停止流程 |
| 产品图片 / 参考图片 | 否 | 有图片时作为产品身份锁定来源；无图片时根据文字生成产品渲染母图 |
| 预设场景 | 否 | 默认 `amazon_9_images_10s_video` |
| 目标平台 | 否 | 随预设自动设置；默认 Amazon |
| 输出比例 | 否 | 商品图默认 1:1；视频默认 9:16；详情页可用 3:4 或 4:5 |
| 视觉风格 | 否 | 默认“高端真实商业摄影，干净奢华，材质准确” |
| 目标客群 | 否 | 默认“礼物购买者与自购女性消费者” |
| 价格带 / 产品等级 | 否 | 默认“中高端轻奢”；不得生成具体价格 |
| 材质与宝石 | 否 | 从描述推断；推断不出时按产品类型使用保守默认 |
| 是否需要模特佩戴 | 否 | 亚马逊/Shopify 默认需要 2-3 张佩戴图；纯白底主图不得出现模特 |
| 视频时长 | 否 | 默认 10 秒；若工具限制或用户改动，可使用 4-15 秒 |
| 视频方向 | 否 | 默认竖屏 9:16；Amazon 横版需要时使用 16:9 |
| 文案语言 | 否 | 默认中文；Amazon 预设默认同时给中文策划稿与英文上架文案 |

若缺少必填字段，调用 `generate_form_for_info_collection` 收集最小信息：

- 只展示缺失必填项。
- 可选项不打断，使用默认值。
- 如果用户选择 `skip`，但必填项仍缺失，说明无法执行并停止。

### 步骤 2：自动补齐默认值

按产品类型补齐珠宝结构细节，保证后续图片与视频不会失真。

| 产品类型 | 默认结构补齐 |
|---|---|
| 戒指 | 戒圈宽度、主石/副石、爪镶或包镶、戒肩、戒托、佩戴舒适度 |
| 项链 / 吊坠 | 链型、链长视觉、吊坠比例、吊环或连接结构、扣头逻辑 |
| 耳环 | 左右成对逻辑、耳针/耳夹/耳钩、垂坠方向、重量感 |
| 手链 / 手镯 | 开口、铰链、扣具、腕部贴合、重复宝石节奏 |
| 胸针 / 配饰 | 背针结构、正面轮廓、佩戴场景、固定方式 |

常用材质默认：

- 铂金 / 白金：冷白色高抛光，适合钻石、蓝宝石、现代婚嫁。
- 18K 黄 / 玫瑰金：温暖奢华，适合珍珠、红宝石、祖母绿、礼物场景。
- 钻石：作为光线控制和高级感点缀，不凭空声明等级。
- 彩宝：只描述视觉色相和切工，不编造产地、证书或净度。

### 步骤 3：轻量确认

如果用户只输入简单需求，例如“蓝宝石戒指，亚马逊九图加 10 秒视频”，不要要求逐项确认，直接执行。

只有以下情况需要先确认：

- 产品类型不明确，例如“这个饰品”但没有图片。
- 用户同时要求互相冲突的平台规格，例如“亚马逊主图但要大量道具和文字”。
- 用户要求可能涉及品牌复刻、证书、价格、医疗功效等高风险内容。

确认话术应简短：

```text
我会按 Amazon 9 图 + 10 秒竖屏视频执行：同一款蓝宝石戒指，白底主图、角度图、细节图、佩戴图、礼盒图、卖点图、英文标题与五点描述，并用生成的产品/佩戴图做参考生成视频。
```

## 三、预设场景

### 3.1 `amazon_9_images_10s_video` 默认预设

适用：Amazon、独立站同步上新、跨境电商素材准备。

交付物：

| 序号 | 物料 | 规格 | 工具 |
|---|---|---|---|
| 1 | 白底主图 | 1:1，纯白背景，产品占画面 80%-90% | `image2image` 或 `text2image` |
| 2 | 45 度角度图 | 1:1，展示结构与宝石火彩 | `image2image` |
| 3 | 侧面 / 背面结构图 | 1:1，展示镶口、戒托、扣具或链路 | `image2image` |
| 4 | 微距细节图 | 1:1，突出宝石切工、金属质感、镶嵌安全 | `image2image` |
| 5 | 模特佩戴图 | 4:5 或 1:1，体现比例与佩戴效果 | `image2image` |
| 6 | 生活方式场景图 | 4:5，礼物、约会、婚礼或通勤场景 | `image2image` |
| 7 | 尺寸 / 结构卖点图 | 1:1，可有简洁图形化布局，但不编造尺寸 | `image2image` |
| 8 | 包装 / 礼盒图 | 1:1，高端礼盒与产品同框 | `image2image` |
| 9 | 品牌氛围图 | 4:5，广告感收尾图，无假 Logo | `image2image` |
| 10 | 10 秒短视频 | 9:16，产品微距、佩戴、礼盒、收尾镜头 | `image2video` / `multi_modal2video` / `video_editor` |
| 11 | 商品文案 | 标题、五点卖点、短描述、视频字幕 | 技能内部生成 |

默认视频镜头：

| 时间 | 画面 | 动作 |
|---|---|---|
| 0-2 秒 | 产品微距主视觉 | 镜头缓慢推进，金属反射流动 |
| 2-4 秒 | 宝石 / 细节 | 轻微旋转或光线扫过，突出切工 |
| 4-6 秒 | 模特佩戴 | 手部、颈部或耳部自然动作，产品清晰 |
| 6-8 秒 | 礼盒 / 场景 | 包装打开或产品置于柔和道具环境 |
| 8-10 秒 | 英雄收尾 | 产品居中，干净背景，保留上架空间 |

### 3.2 `shopify_pdp_7_images_8s_video`

适用：Shopify、品牌官网 PDP。

交付物：

- 7 张图：主图、角度、微距、模特、生活方式、包装、品牌氛围。
- 8 秒视频：更偏品牌质感，少文字，重镜头流动。
- 文案：产品标题、短描述、材质说明、护理提示、SEO 摘要。

默认调用链：

`text2image` 或 `image2image` → `image2image` 扩展 7 图 → `image_super_resolution` → `multi_modal2video` → `video_editor`

### 3.3 `tmall_detail_page`

适用：天猫、淘宝、京东详情页长图。

交付物：

- 8-12 个详情页模块：首屏海报、材质模块、佩戴模块、细节模块、礼赠模块、售后/护理模块。
- 3:4 或 4:5 图片组。
- 中文主标题、卖点短句、模块标题与详情页文案。

默认调用链：

`image2image` → `foreground_segmentation` → `image_super_resolution` → 技能内部生成详情页文案

### 3.4 `xiaohongshu_douyin_seed`

适用：小红书封面、抖音种草短视频、信息流素材。

交付物：

- 3 张种草封面图：佩戴、生活方式、细节。
- 1 条 10-15 秒竖屏短视频。
- 口播脚本、字幕条、标题建议、封面标题。

默认调用链：

`image2image` → `multi_modal2video` → `video_editor`

### 3.5 `clean_product_pack`

适用：已有产品图需要电商标准化。

交付物：

- 去背景白底图。
- 高清主图。
- 角度补充图。
- 简短上架文案。

默认调用链：

`image2image` → `foreground_segmentation` → `image_super_resolution`

## 四、执行阶段

执行阶段自主完成，不再让用户选择工具。

### 步骤 4：建立产品身份锁

在任何生成前先写出“产品身份锁”，后续每张图和每条视频都必须遵守。

产品身份锁包括：

- 产品类型：戒指、项链、耳环、手链、吊坠等。
- 主轮廓：整体形状、比例、视觉重心。
- 材质：金属颜色、抛光/拉丝/哑光/珐琅等。
- 宝石方案：主石位置、颜色、切工、副石排列。
- 结构：镶口、爪数、链条、扣具、耳针、铰链、开口等。
- 禁止变化：不得增加无关宝石、改换金属、改变主石形状、添加假品牌标识。

如果用户上传产品图：

- 产品图优先级最高。
- 所有商品图用 `image2image` 生成。
- 视频用生成后的主图、佩戴图或场景图作为参考，不用纯文本直接生成。

如果用户没有上传产品图：

- 先用 `text2image` 生成一张“产品母图”。
- 通过质量检查后，将产品母图作为后续 `image2image` 的产品身份参考。
- 交付中标注为“基于文字设计生成的产品渲染”，不声称真实 SKU 完全一致。

### 步骤 5：图片生成

#### 5.1 无产品图时

调用 `text2image`：

```text
工具: text2image
目标: 生成 1 张产品母图
提示词必须包含:
- 珠宝类型、材质、宝石、结构、比例
- 专业电商摄影视角
- 干净背景、真实金属反射、可信宝石折射
- 无文字、无 Logo、无证书、无价格
```

母图通过检查后，后续图片调用 `image2image`。

#### 5.2 有产品图或母图后

按预设槽位逐张调用 `image2image`：

```text
工具: image2image
输入: 产品图 / 产品母图 / 已确认角度图
目标: 生成对应槽位的商品图
提示词必须包含:
- 产品身份锁
- 当前槽位目的
- 摄影构图、背景、灯光、比例
- 禁止改变产品结构
```

白底主图约束：

- 纯白或接近纯白背景。
- 无道具、无文字、无水印、无模特。
- 产品完整，不裁切，不变形。
- 产品占画面主体区域，边缘保留安全留白。

佩戴图约束：

- 模特不抢产品，珠宝必须清晰可见。
- 不出现其他竞争首饰，除非用户要求套装。
- 手、脖颈、耳部动作自然，比例可信。
- 肤色、服装、背景服务于产品风格。

细节图约束：

- 微距清晰，宝石切面和金属边缘要可读。
- 大颗宝石必须有可见支撑。
- 链条、扣具、爪镶、包镶等结构不能凭空消失。

### 步骤 6：去背景与高清处理

对需要电商标准化的图执行：

```text
工具: foreground_segmentation
适用:
- 白底主图需要更干净边缘
- 需要透明底素材
- 详情页需要产品抠图组合
```

然后调用：

```text
工具: image_super_resolution
适用:
- 主图、细节图、详情页首屏、视频关键帧
- 任何边缘不够锐利或宝石细节不够清晰的图
```

不要对已经有明显结构错误的图片直接超分；应先重生成或修正。

### 步骤 7：文案生成

文案由技能内部生成，不需要单独工具。

Amazon 默认文案结构：

| 模块 | 输出要求 |
|---|---|
| 英文标题 | 150-190 字符，包含产品类型、核心材质、场景，不写虚假认证 |
| Five Bullets | 5 条，每条 120-220 字符，强调设计、材质、佩戴、礼赠、保养 |
| Short Description | 1 段 80-120 词，适合 PDP 开头 |
| Image Captions | 9 条，每张图 1 句，简短说明用途 |
| Video Script | 10 秒镜头脚本、字幕、可选旁白 |
| 中文策划说明 | 给运营团队看的中文卖点摘要 |

禁止文案：

- “天然”“无瑕”“顶级”“保值”“治疗”“官方认证”等未经用户提供证据的声明。
- 具体克拉、尺寸、重量、证书号、价格、产地。
- 假品牌名、假 Logo、侵权性系列名。

### 步骤 8：视频生成

视频必须优先使用已经生成或用户上传的图片作为参考。

选择工具：

| 条件 | 工具 |
|---|---|
| 只有 1 张高质量主图，要做简单动态展示 | `image2video` |
| 有首帧和尾帧，需要明确开合、远近、转场 | `start_end2video` |
| 有多张商品图 / 佩戴图 / 礼盒图，要串成 10 秒广告 | `multi_frame2video` |
| 有图片、视频、音频等多模态参考，需要更强一致性 | `multi_modal2video` |
| 多段视频需要拼接、加音乐或整理最终成片 | `video_editor` |

默认 10 秒视频建议使用 `multi_modal2video` 或 `multi_frame2video`。

视频提示词必须包含：

- 产品身份锁，并说明参考图控制珠宝外观。
- 时长：默认 10 秒。
- 画幅：默认 9:16，平台需要时可改 1:1 或 16:9。
- 镜头顺序：微距、产品旋转、佩戴、礼盒/场景、英雄收尾。
- 运动：慢推、轻微环绕、光线扫过，不做夸张变形。
- 负面约束：不改变首饰结构、不添加多余宝石、不出现假 Logo、不遮挡产品。

示例视频提示词结构：

```text
使用 @图片1 作为珠宝产品身份参考，@图片2 作为佩戴比例参考，生成 10 秒 9:16 高端珠宝电商短视频。
镜头 1（0-2s）：微距展示主石和金属反射，镜头缓慢推进。
镜头 2（2-4s）：产品轻微环绕，宝石火彩自然闪动，结构保持不变。
镜头 3（4-6s）：模特佩戴场景，动作优雅克制，珠宝始终清晰。
镜头 4（6-8s）：礼盒或柔和场景，突出送礼氛围。
镜头 5（8-10s）：产品居中英雄收尾，背景干净，留出字幕空间。
禁止改变珠宝造型、宝石颜色、镶嵌结构、金属材质；禁止文字水印、假 Logo、额外首饰。
```

视频生成后如需要合成：

```text
工具: video_editor
目标:
- 拼接多个片段
- 加入轻音乐
- 整理为一个最终成片
- 保持画幅、节奏和产品清晰度
```

## 五、质量检查与重试

每次生成后按以下标准检查。

### 5.1 图片质量标准

| 编号 | 检查项 | 通过标准 |
|---|---|---|
| JI-1 | 产品一致性 | 产品类型、主石、金属、结构与身份锁一致 |
| JI-2 | 结构可信 | 大石有支撑，链条/扣具/戒托/耳针逻辑合理 |
| JI-3 | 电商可用 | 主体清晰，构图符合平台用途 |
| JI-4 | 细节清晰 | 宝石切面、金属边缘、镶口可读 |
| JI-5 | 合规安全 | 无假证书、假 Logo、价格、夸大功效 |
| JI-6 | 槽位匹配 | 每张图完成自己的电商目的，不重复 |

不通过时：

- 若产品结构错：补充产品身份锁，调用 `image2image` 重生成。
- 若背景不干净：调用 `foreground_segmentation` 或重做白底主图。
- 若细节不清：先判断结构是否正确；正确则调用 `image_super_resolution`。
- 每个槽位最多重试 3 次；仍失败时标注为待人工复核，不假装完成。

### 5.2 视频质量标准

| 编号 | 检查项 | 通过标准 |
|---|---|---|
| JV-1 | 产品锁定 | 视频中珠宝外观不漂移，不改变主石和结构 |
| JV-2 | 可售卖性 | 画面清楚展示产品，不只做氛围镜头 |
| JV-3 | 运动克制 | 镜头自然高级，无夸张变形 |
| JV-4 | 平台适配 | 时长、比例、节奏符合预设 |
| JV-5 | 合规 | 无假品牌、假证书、虚假功效和不可控文字 |

不通过时：

- 若产品漂移：改用更多参考图，优先 `multi_modal2video`。
- 若镜头太乱：缩短镜头动作描述，减少复杂转场。
- 若产品不清楚：在提示词中强调“珠宝始终清晰可见，占据画面视觉中心”。
- 最多重试 2 次；仍失败时交付已通过的图片和文案，并标注视频阻塞原因。

## 九、执行提醒

- 默认尽量一步到位，不把可选项变成问卷。
- 每一张图都要重复产品身份锁，防止生成过程中漂移。
- 有产品图时优先 `image2image`，无产品图时先 `text2image` 生成母图。
- 视频要参考已生成或上传的图片，不要让视频从纯文本重新发明珠宝。
- 交付文案必须保守、可上架，不编造证明型信息。
- 任何未通过质量检查的槽位都应明确标注状态，不能假装完成。
