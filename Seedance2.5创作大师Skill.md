---
技能名称: Seedance 2.5 创作大师Skill
技能描述: Seedance 2.5 视频创作与可控编辑复合技能。用于 30 秒连贯直出、长镜头叙事、多参考图视频、参考视频编辑、多语种文字视频、产品广告、品牌概念片、教程科普、动作短片、非遗人文、奇幻科幻、视觉特效和用户已有视频局部改造。用户给一句创意、上传参考图/视频/音频或要求使用 Seedance 2.5 时，先把需求整理成可执行视频 brief，再生成 Seedance 2.5 prompt，调用视频工具，最后完成视频 QA、修复和交付。
---

技能内容：

分为**交互阶段**和**执行阶段**。交互阶段把用户需求规范化为 Seedance 2.5 视频 brief；执行阶段按“素材分析 / 参考素材生成或绑定 → Seedance 2.5 prompt → 视频生成或视频编辑 → 视频 QA / 修复 → 最终交付”的链路执行。

核心原则：

- 用户要求视频时，不能停在创意方向、分镜、prompt 或素材清单，必须继续调用视频工具。
- Seedance 2.5 默认支持更长的连贯叙事，默认目标时长为 30 秒；用户明确指定时长时按用户指定。
- 长视频必须使用时间轴拆解，每 2-5 秒设置一个清晰 beat。
- 多参考输入必须给每个参考素材分配单一角色，避免身份、场景、动作和风格互相污染。
- 可控编辑任务必须先写保持项，再写变化项，最后写时序一致和边缘自然约束。

## 一、触发范围

### 1.1 应使用本技能的情况

- 用户明确提到 Seedance 2.5、seedance2.5、2.5 视频、30 秒直出、长镜头、一镜到底或多参考视频。
- 用户要求生成完整视频：品牌概念片、广告片、产品视频、教程视频、科普视频、剧情短片、动作短片、人文纪录感短片、视觉特效短片、文字动画、多语种视频。
- 用户上传多张参考图，希望按人物、环境、风格、镜头、文字、产品、声音分别控制视频。
- 用户上传参考视频，希望保持原视频的人物、动作、构图、镜头运动或节奏，只做新增、删除、替换、风格化或特效增强。
- 用户需要 26-33 秒级别的连贯叙事、跨场景转场、时间流逝、四季变换、全球传递、教程步骤或多语种文字展示。

### 1.2 不应使用本技能的情况

- 用户只要静态图片、海报、封面、珠宝设计图或纯文案，不需要视频。
- 用户只要 15 秒内普通影视镜头，且已经明确使用 Seedance 2.0 或现有短镜头流程。
- 用户的核心需求是大批量任务数量守恒、分批、并发、重试和清单管理；此时先用批量规划技能，再把单条视频任务交给本技能。
- 用户要求违法危险行为、误导性真实人物发言、露骨色情、仇恨骚扰或其他不适合生成的视频内容。

## 四、交互阶段

### 4.1 信息提取

| 字段 | 必填 | 默认值 | 缺失时处理 |
|---|---|---|---|
| 任务类型 | 是 | 自动路由 | 无法判断时按 `cinematic_creator` |
| 核心主体 | 是 | 无 | 没有主体且无法从素材推断时调用 `generate_form_for_info_collection` |
| 视频目标 | 是 | 生成 1 条完整 Seedance 2.5 视频 | 从用户语义推断 |
| 时长 | 否 | 30 秒 | 用户指定时按指定；短任务可用 8-15 秒 |
| 画幅 | 否 | 16:9 | 短视频、口播、社媒默认 9:16；方形内容默认 1:1 |
| 参考素材 | 否 | 无 | 有素材则分析用途并绑定角色 |
| 参考素材角色 | 否 | 自动分配 | 多素材时必须输出绑定表 |
| 视觉主线 | 是 | 自动提炼 | 从主体、物件、角色、文字或动作中选择一个贯穿主线 |
| 时间轴密度 | 否 | 每 2-5 秒一个 beat | 教程、科普、多语种文字按步骤或语言拆分 |
| 镜头策略 | 否 | 稳定跟拍 + 清晰转场 | 从任务类型推断 |
| 转场方式 | 否 | 主线物件转场 / match-cut / 前景遮挡 / 运动模糊 | 按场景选择 |
| 文字与语言 | 否 | 无文字 | 用户要求文字时必须记录准确文本、语言和出现时间 |
| 声音 | 否 | 可选环境音 / 音效 / 音乐节奏 | 用户给音频或节奏时绑定为参考 |
| 风险点 | 否 | 自动识别 | 多人物、文字、品牌、手部、复杂动作、长镜头、低光、水下、特效 |
| 输出目标 | 否 | 完整视频 + prompt + QA 摘要 | 用户只要 prompt 时只输出 prompt |
| 视频模型 | 否 | Seedance 2.5 | 用户指定时按指定 |

### 4.2 必要追问规则

只在以下情况追问：

- 没有主体，也没有任何可分析参考素材。
- 用户要求准确文字、logo、字幕或多语种内容，但没有给出准确文本。
- 用户要求还原具体产品、角色、场地或视频，但没有提供足够参考素材。
- 用户要求多段成片、平台适配或商业交付，但缺少最终画幅或平台要求且无法推断。

其他字段使用默认值继续执行。

## 六、Prompt 构建规则

### 6.1 Seedance 2.5 Prompt 骨架

```text
任务目标：
生成一条 {duration} 秒 {aspect_ratio} 的 Seedance 2.5 视频，类型为 {task_type}。

视觉主线：
用 {visual_throughline} 贯穿全片。该主线在不同场景、时间、风格或动作中保持可识别，不丢失、不随机变形。

参考绑定：
{reference_binding_table}

整体风格：
{style}，{lighting}，{color_palette}，{material_quality}，{camera_texture}。

时间轴：
{timeline_beats}

镜头与运动：
{camera_strategy}
镜头运动必须有动机，和主体动作、场景变化或音乐节奏匹配。

转场：
{transition_strategy}
转场要服务于视觉主线，避免无意义跳切。

声音 / 文字：
{audio_text_requirements}

负向约束：
不要改变核心主体身份。不要混用参考素材角色。不要出现随机文字、错误语言、无关 logo、水印、UI、边缘涂抹、闪烁、重影、角色漂移、产品变形、动作断裂或不受控镜头晃动。
```

### 6.2 时间轴写法

| 视频类型 | 时间轴写法 |
|---|---|
| 30 秒叙事 | `0-5 秒 / 5-10 秒 / 10-15 秒 / 15-20 秒 / 20-25 秒 / 25-30 秒` |
| 26 秒长镜头 | `0-3 秒` 到 `24-26 秒`，每段写镜头方位和空间变化 |
| 15 秒文字动画 | 每种语言约 1-1.5 秒，逐条写文字、材质、转场 |
| 教程视频 | 按步骤写时间、景别、位置、动作、画面要求和解说 |
| 动作视频 | 开场对峙、第一次冲突、低谷、反击、高潮、余韵 |
| 可控编辑 | 不重写全片时间轴，只写保持项、变化发生时机和变化范围 |

### 6.3 参考绑定规则

```Markdown
| 素材 | 角色 | 只继承 | 不继承 |
|---|---|---|---|
| @图像1 | 主体身份 | 轮廓、脸、产品形态、材质、颜色 | 背景、随机文字 |
| @图像2 | 环境空间 | 空间结构、光线、色彩、氛围 | 人物身份、产品形态 |
| @图像3 | 动作 / 风格 | 动作方向、镜头节奏、风格语言 | 人物身份、品牌文字 |
| @视频1 | 原始视频保持项 | 人物、动作、镜头、构图、节奏、时长 | 需要替换或删除的对象 |
| @音频1 | 声音参考 | 音效、节拍、情绪 | 画面内容 |
```

多参考素材必须写明“只继承”和“不继承”。没有绑定表时，不得调用多模态视频工具。

### 6.4 可控编辑 Prompt 模板

```text
保持 {reference_video} 中的 {人物 / 产品 / 环境 / 镜头运动 / 构图 / 动作节奏 / 时长} 不变。

编辑目标：
{新增 / 删除 / 替换 / 风格化 / 特效增强} {target_object_or_effect}。

变化细节：
{appearance}，{material}，{motion}，{timing}，{interaction_with_original_scene}。

时序一致：
变化必须跟随原视频动作和镜头运动，前后帧连续，不闪烁，不跳动，不出现边缘涂抹、重影、形变或遮挡错误。

负向约束：
不改变原人物身份、环境、构图、镜头节奏和未指定对象。
```

### 6.5 多语种文字 Prompt 规则

- 每个语言只出现该语言对应文字，不混排、不叠加、不新增其他语言。
- 每个语言独立时间段，写清出现方式、材质、背景、转场和消失方式。
- 字符必须逐字准确；不确定的文字先追问。
- 文字在自然场景中形成时，要写形成介质，例如云雾、水波、灯带、草地、玻璃反射、喷泉水雾。
- 文字动画要写字体、材质、运动、粒子、溶解、循环闭合方式。

```Markdown

- planned_count:
```

## 九、质量门

### 9.1 brief_intake

| 检查项 | 标准 |
|---|---|
| 主体 | 有明确核心主体或可从素材识别 |
| 目标 | 明确是生成、编辑、替换、删除、教程、科普、广告或文字视频 |
| 时长 | 有默认或用户指定时长 |
| 参考素材 | 多素材已绑定角色 |
| 风险 | 已识别文字、人物、产品、复杂动作、长镜头、特效等风险 |

### 9.2 reference_asset

| 检查项 | 标准 |
|---|---|
| 身份 | 主体清晰、可作为视频参考 |
| 单一角色 | 每个素材只承担指定作用 |
| 画面 | 无多余文字、水印、UI、错误 logo |
| 连续性 | 关键帧或起止帧的主体、光线、空间方向一致 |

### 9.3 seedance25_prompt

满分 100，低于 88 必须重写。

| 项目 | 分值 | 标准 |
|---|---:|---|
| 目标清晰 | 15 | 视频类型、时长、画幅、模型目标明确 |
| 视觉主线 | 15 | 有贯穿全片的主体、物件、动作或文字主线 |
| 时间轴 | 20 | 每个 beat 有时间、场景、动作、镜头和变化 |
| 参考绑定 | 15 | 每个参考素材职责清楚，不混用 |
| 镜头与转场 | 15 | 运镜有动机，转场可执行 |
| 声音与文字 | 10 | 文本准确，声音和节奏约束清楚 |
| 负向约束 | 10 | 针对漂移、闪烁、错误文字、变形等风险 |

### 9.4 video_output

| 检查项 | 通过标准 |
|---|---|
| 完整性 | 达到目标时长和主要 beat |
| 主线一致 | 核心主体、物件或文字线索贯穿 |
| 参考遵循 | 多参考没有互相污染 |
| 镜头稳定 | 无随机晃动、跳切、方向混乱 |
| 文字准确 | 多语种、品牌字、字幕不乱码、不混排 |
| 编辑自然 | 删除、替换、新增效果边缘自然、时序连续 |
| 视觉质量 | 无明显闪烁、重影、肢体异常、产品变形 |

### 9.5 final_delivery

- 最终视频可播放。
- 用户要求的画幅、字幕、配乐、片头片尾或平台版本已完成。
- 数量汇总无缺失。
- 失败与修复记录清楚。

## 十、失败修复

| 失败类型 | 诊断 | 修复动作 |
|---|---|---|
| 长镜头断裂 | 时间轴跨度大或转场动机弱 | 缩短 beat、增加视觉主线、改用递物 / 前景遮挡 / match-cut |
| 主体漂移 | 参考绑定不清或主体过多 | 强化 identity_lock，减少无关角色 |
| 多参考污染 | 图像角色混用 | 重写参考绑定表，写明只继承和不继承 |
| 文字错误 | 文字太多或语言混排 | 每段只保留一个准确文本，降低同时出现数量 |
| 编辑闪烁 | 变化范围过大或边缘约束不足 | 明确保持项，限制变化区域，加入时序一致约束 |
| 产品变形 | 动作过强或材质描述不足 | 增加产品形态、比例、材质、logo 位置锁 |
| 动作混乱 | 高速动作太多 | 保留 1-2 个主动作，用慢动作高光替代密集招式 |
| 风格不统一 | 场景风格跨度过大 | 使用同一色彩、镜头质感、主线物件和声音节奏收束 |

修复时保留原 job_id，追加版本号，例如 `SD25-V01-R1`。默认最多重试 2 次；关键交付最多 3 次。
